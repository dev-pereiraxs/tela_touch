# Touch Screen Interface

Uma interface touch moderna desenvolvida com **HTML, CSS, JavaScript, Python e Django**, focada em usabilidade, desempenho e integração com um servidor dedicado.

Este projeto faz parte de uma solução maior e atua como a camada de interação com o usuário, comunicando-se com um projeto de servidor hospedado em um repositório separado.

### 🚀 Stack

* HTML5
* CSS3
* JavaScript
* Python
* Django

> **Nota:** Este repositório contém apenas a interface touch. O backend/servidor encontra-se em outro repositório do GitHub.
